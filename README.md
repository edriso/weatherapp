# Weather App

A simple weather app project using Express.js, and Ejs

### Project setup

- Create .env file, then add APIKEY variable equal to key value; _which you can get from [openweathermap](https://openweathermap.org/)_
- Run `npm install`

### Run project

```
npm start
```
